# Agroecosystems Analysis (Rickerl & Francis, eds., 2004)

ASA–CSSA–SSSA Agronomy Monograph No. 43. 211 pages, 12 chapters. Multi-author; each chapter is a self-contained essay. Use chapter number + author name as the citation anchor (page numbers are unreliable from pdftotext output; grep `ricker_francis_full_text.md` for exact passages).

**Stance:** Complements Gliessman by treating the agroecosystem *as a system* — emphasizes hierarchy of spatial scales (field → farm → landscape → region), whole-farm planning, multifunctional economic/social analysis, and ethics. Use this corpus when critique needs a systems/landscape framing that Gliessman under-specifies.
